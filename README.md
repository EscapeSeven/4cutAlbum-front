# 네컷앨범

> 인생네컷을 찍고나서 서랍장에 처박아두는 사진들 아쉽지 않으세요?

**네컷앨범**은 인생네컷의 QR CODE 찍으면 사진/영상을 저장할 수 있고 그 사진들로 앨범을 만들 수 있습니다. <br/><br/>
**네컷앨범**은 사진을 스티커로 꾸미고 앨범을 친구들에게 공유할 수 있습니다. <br/><br/>

🔗 서비스 링크 : https://4cus.vercel.app/
